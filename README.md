# Programação Orientada a Objetos (POO)

Repositório dedicado às atividades da disciplina de **Programação Orientada a Objetos (POO)** do curso de **Ciência da Computação**.  
Contém exercícios e pequenos projetos desenvolvidos ao longo da matéria, com foco na prática de conceitos fundamentais da programação orientada a objetos.

## 📚 Conteúdos abordados

Durante o desenvolvimento das atividades, foram trabalhados conceitos como:

- Classes e Objetos
- Encapsulamento
- Métodos
- Construtores
- Organização do código
- Modelagem básica de sistemas orientados a objetos

## 💻 Tecnologias utilizadas

- Java
- NetBeans (IDE utilizada nas atividades)

## 📁 Estrutura do repositório

Cada pasta ou arquivo representa uma atividade ou exercício realizado durante a disciplina.
